# The Fit Club

**The Fit Club** is a modern fitness web application built with **React**, **TypeScript**, and **ShadCN UI**. It offers membership management, workout tracking, and wellness tips for users. The app uses **Redux** for state management, **Axios** for API requests, **React Router DOM** for routing, and **Clerk** for authentication.

---

## **Features**

- User authentication via **Clerk**
- Membership management and subscription tracking
- Personalized workout plans
- Wellness tips carousel
- Progress tracking using animated **Progress Rings**
- Modular and reusable **ShadCN UI** components
- State management using **Redux**
- Smooth navigation with **React Router DOM**
- API integration using **Axios**

---

## **Tech Stack**

- **React 18** + **TypeScript**
- **ShadCN UI** for design system and components
- **Redux** for global state management
- **Axios** for HTTP requests
- **React Router DOM** for client-side routing
- **Clerk** for authentication
- **PNPM** as the package manager
- **Framer Motion** for animations

---

## **Getting Started**

### **Prerequisites**

- Node.js v18+
- PNPM package manager installed globally
- A Clerk account (for authentication)
